This is a map visualization of the Natural Earth Data. These guys are great. If it is not working for you for any reason, the best thing to do is to get a newer and/or better browser. It should work in the latest versions of Google Chrome, Firefox, Safari, and Internet Explorer. If it still isn't working, especially if it simply is not loading some of the denser data, this is a fundamental problem with your browser or operating system which doesn't allow it to evaluate this quantity of data and there isn't much I can do for you, sorry. If you zoom in somewhat, this application will load something north of 100 Mb of data defining over 30,000 polygons and 5 million points. This is a lot of data!

The code is open source under an MIT License and is hosted at code.google.com/p/ned-map. Natural Earth Data is in the public domain. Parts of the rendering were inspired by shapefile-js (github.com/RandomEtc/shapefile-js), which is a great demonstration of the same principles. The primary difference is that I preprocess the shapefiles behind the scenes and load pure JSON instead.
